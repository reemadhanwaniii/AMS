# FlightService

## Project setup 

- clone project
- execute npm install
- configure env variables
- inside src/config create config.json

```
    "development": {
    "username": "username"
    "password": "pswd",
    "database": "Flights_Search_DB_DEV",
    "host": "127.0.0.1",
    "dialect": "mysql"
  },
```


execute npx sequelize db:create
